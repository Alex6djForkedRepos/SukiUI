# Slider

A component for displaying current value and intervals in range.

## Show

<img src="/controls/inputs/slider.gif"/>

## Example

```xml
<Slider IsSnapToTickEnabled="True"
        Maximum="100"
        Minimum="0"
        TickFrequency="1"
        Value="{Binding}" />
```
